# Fragrance of Mastership

This is a mobile application built for Android devices that provides a collection of authentic Hadiths from the Islamic religion. The app features a simple and intuitive user interface that allows users to easily browse through the Hadiths and read them in three different languages: Arabic, English, and Urdu.

## Features

- Access to a large collection of authentic Hadiths.
- Hadiths available in three languages: Arabic, English, and Urdu.
- Simple and intuitive user interface for easy navigation.

## Technologies Used

- Flutter SDK
- Dart programming language
- Firebase Authentication
- Firebase Realtime Database
- Git and GitHub for version control

## Installation

To install and run this app on your Android device, follow these steps:

1. Clone this repository to your local machine.
2. Open the project in Android Studio.
3. Connect your Android device to your computer.
4. Build and run the app on your device using Android Studio.
